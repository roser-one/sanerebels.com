## Usage Guidelines

1. **Logo Clear Space:** Maintain minimum clear space equal to the height of the "S" in SANE
2. **Minimum Size:** Logo should be at least 120px wide for digital, 30mm for print
3. **Background:** Prefer dark backgrounds; use light logo variants on light backgrounds
4. **Animation:** The logo animation (SANE/REBELS alternating) is for web only
5. **No Modifications:** Do not stretch, rotate, or alter the logo colors
